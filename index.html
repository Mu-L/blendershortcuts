Blender Shortcuts

If want to become lightning-fast in blender, there is no way around using shortcuts. This website is designed to help you learn the most common ones.

A keyboard with a numpad and a mouse with a clickable wheel is highly recommended!

Change View

Middle Mouse

  • Drag your Middle Mouse or Mouse Wheel to rotate the view. It rotates around an invisible point of interest.

  • Hold Shift and drag to move the point of interest.

  • Hold Ctrl and drag or roll your Mouse Wheel to zoom in and out

Axis View

Numpad 1 – 9

  • Num 1, Num 3 & Num 7 for front view,side view & top view.

  • Num 9 to invert the current view direction.

  • Num 5 to toggle orthographic view.

Camera View

Numpad 0

  • Numpad 0 to align the view with the scene camera.

  • Ctrl + Numpad 0 to align the view with the selected camera and make it the scene camera.

  • Ctrl + Alt + Numpad 0 to align the scene camera with the current view.

Frame Selected

Numpad .

  • Numpad . to set the point of interest to the current selection. This makes it easier to navigate and can fix issues with zooming.

  • It will set the point of interest to the average position of your selection.

Change Mode

Ctrl + Tab

  • Ctrl + Tab to show you all available modes.

  • Tab to toggle Edit mode directly.

  • If your mouse is hovering a timeline, Ctrl + Tab to toggle graph view.

Move

G

  • Press X, Y or Z to lock to an axis. Press twice to switch from global to local axes.

  • Shift+X, Shift+Y, Shift+Z to exclude an axis.

  • In Edit mode, press G twice to slide along existing edges.

Rotate

R

  • Press X, Y or Z to lock to an axis. Press twice to switch from global to local axes.

  • Shift+X, Shift+Y, Shift+Z to exclude an axis.

  • Press R twice to switch to gimbal mode.

Scale

S

  • Press X, Y or Z to lock to an axis. Press twice to switch from global to local axes.

  • Shift+X, Shift+Y, Shift+Z to exclude an axis.

Delete

X

  • Press X in Object Mode to delete selected objects.

  • Press X in Edit Mode to delete selected vertices, edges & faces.

Hide

H

  • H to hide the selection.

  • Shift + H to hide everything but the selection.

  • Alt + H to unhide all.

  • Meshes that are in Edit Mode are visible in Object Mode.

Add

Shift + A

  • Shift + A adds new data, depending on the context.

  • Adds an object in Object Mode, additional data to the active object in Edit Mode or a new node in one of the Node Editors

Duplicate

Shift + D

  • Shift + D to duplicate the selection.

  • In Object Mode, Alt + D creates linked duplicates which reference the original data instead of copying it.

Change Shading

Z

  • Shift + Z to toggle wireframe and X-ray mode, to see and select through the mesh.

  • Alt + Z to toggle X-ray mode.

Snap

Shift + S

  • Move one thing precisely to another, especially useful to position the 3D cursor.

  • Shift + Right Mouse to set the 3D cursor by hand.

Select All

A

  • A to select all objects, meshes, keyframes etc. depending the context.

  • Alt + A to deselect all.

Add to Selection

Shift + Left Mouse

  • Shift + Left Mouse on unselected to add to the selection.

  • Shift + Left Mouse on selected to remove from the selection.

Select Loop

Alt + Left Mouse

  • Multiple four-sided faces ("quads") in a row make a loop. To select a loop, press Alt + Left Mouse while your mouse is close to a crossing edge.

  • Alt+Shift+Left Mouse to add or remove loops from the selection.

Select Shortest Path

Ctrl + Left Mouse

  • Ctrl+Left Mouse to add the shortest path of faces to the selection.

  • Ctrl+Shift+Left Mouse to add or remove an area.

Select Linked

L

  • Hover a mesh and press L to select all linked vertices.

  • Hover a mesh and press Shift + L to remove all linked vertices from the selection

  • Ctrl + L to select what is linked to your current selection.

Extrude

E

  • E to extrude a connected copy of the selected mesh and start a move.

  • Escape or Right click to cancel the move, but not the extrude.

  • Alt + E to extrude along face normals.

Inset

I

  • I to inset around the selected faces.

  • Press I again to inset individual faces.

  • Press O to toggle outset.

  • Press B to toggle boundary.

Merge

M

  • Merge all selected vertices at a specified point.

  • Merge By Distance to remove vertices that are closer than the threshold value, which you can set in the context menu.

Add Face

F

  • F to add a new face to the mesh. Adds a loose edge if two seperate vertices are selected.

  • Alt + F to fill Ngons with triangles.

  • Ctrl + F to bring up all face options including "Grid Fill"

Loop Cut

Ctrl + R

  • Ctrl + R to cut a loop if faces into segments.

  • Roll your Mouse Wheel to change the number of segments.

  • You can only cut multiple faces in a row if they are four-sided ("quads").

Bevel

Ctrl + B

  • Ctrl + B to bevel edges.

  • Ctrl + Shift + B to bevel vertices.

  • Roll your Mouse Wheel to adjust the number of segments.

  • V to switch between vertex and edge mode.

  • C to toggle clamping.

  • P to adjust the bevel profile.

Dissolve

Ctrl + X

  • Ctrl + X to delete vertices but connect the surrounding edges. The result is based on your current selection mode (vertex, edge, face).

  • X > Limited Dissolve dissolves all vertices/edges below a treshold angle.

Shrink/Fatten

Alt + S

  • Alt + S to move all vertices along their vertex normals.

Snap (Toggle)

Ctrl

  • Hold Ctrl to enable snapping.

  • Works with Move, Rotate and Scale as well as Sliders

  • You can snap to increments (default) or geometry (vertices, edges, faces).

Granular Input

Shift

  • Hold Shift to divide your mouse input for more precise control.

  • Works with Move, Rotate and Scale as well as Sliders

Multi-Edit

Alt

  • If you have multiple objects selected, editing parameters will only affect the active object.

  • Hold Alt to change the parameters of all selected objects instead.
